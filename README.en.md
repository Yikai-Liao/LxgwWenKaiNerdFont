# LXGW WenKai Nerd Font

LXGW WenKai patched with Nerd Font glyphs.

中文说明： [README.md](README.md)

## Fonts

- Proportional: `LXGWWenKai Nerd Font`
- Monospace: `LXGWWenKaiMono Nerd Font`
- Weights: `Light`, `Regular`, `Medium`

## Install

### AUR

```bash
paru -S ttf-lxgw-wenkai-nerd
paru -S ttf-lxgw-wenkai-mono-nerd
```

- `ttf-lxgw-wenkai-nerd`: proportional
- `ttf-lxgw-wenkai-mono-nerd`: monospace

### Homebrew

```bash
brew tap Yikai-Liao/fonts
brew install --cask font-lxgw-wenkai-nerd
brew install --cask font-lxgw-wenkai-mono-nerd
```

### Manual

1. Download the required asset from [Releases](https://github.com/Yikai-Liao/LxgwWenKaiNerdFont/releases).
2. Available assets:
   - `lxgw-wenkai-nerd-<version>.zip` / `.tar.gz`: full bundle
   - `lxgw-wenkai-nerd-proportional-<version>.zip`: proportional only
   - `lxgw-wenkai-nerd-mono-<version>.zip`: monospace only
3. Extract and install the `.ttf` files:
   - Linux: `~/.local/share/fonts/` or `/usr/share/fonts/`
   - macOS: install with Font Book
   - Windows: right-click and install
